## How to open the project 
1. **clone the repository on your local desk then open the project directory**
```
git clone https://github.com/Marya-Alzubi/Plates-Co
```
```
cd Plates-Co
```
2. **download the packages**
```
npm install
```

3. **open the project**
```
npm start
```
